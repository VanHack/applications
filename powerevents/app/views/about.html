PowerEvents Application

PowerEvents is an application to show you informations about your favorites artists. Our application is based on REST API concepts and we find the searchs getting the Bandsintown REST API results.

This application was made for the VanHackaton 4.0. This application was created using Yeoman with Angular-cli and the technologies bellow was used:
